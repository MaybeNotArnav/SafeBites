class AllergenInfo(BaseModel):

View In Hierarchy

Represents metadata about a potential allergen found in a dish. This model provides structured information on allergens either explicitly declared or inferred from ingredients, including confidence levels and reasoning. Attributes: allergen (str): The name of the allergen (e.g., "peanuts", "dairy"). confidence (Optional[float]): A numeric confidence score (0.0–1.0) indicating certainty about the allergen's presence. why (Optional[str]): An optional explanation describing how or why this allergen was detected or inferred.
